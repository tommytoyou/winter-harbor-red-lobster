# Winter Harbor Red Lobster

Premium wild-caught Maine red lobster brand website built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

- `/` — Home (luxury single-scroll experience)
- `/our-story` — Brand heritage
- `/shop` — Product collection
- `/the-harvest` — From trap to table

## Stack

- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
